# Укорачиватель ссылок

## Info
Есть всего один хендлер ```/```, который и выполняет всю основную логику.
При **POST** запросе с формой ```url=someURL``` в бд добавляется оригинальная ссылка и укороченная. Укорченная ссылка получается путем её **хеширования** (исполюзуется FNV64). При **GET** запросе программа обращается к бд и при нахождении возвращает исходную ссылку в **JSON** формате.
Также и ошибки(несуществующие ссылки и тд) приходят в **JSON** формате.

При запуске можно указать флаг для выбора хранилища (Postgres или in-memory). Например:
```bash
    $ mkdir bin
    $ make build
    $ ./bin/ozon-task.exe -store_type in-memory # запускаем in-memory вариант
```
или запустить без флага и будет поствален Postgres, который по умолчанию.
```bash
    $ mkdir bin
    $ make build
    $ ./bin/ozon-task.exe # запускаем Postgres вариант
```
